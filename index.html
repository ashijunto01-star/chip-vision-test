色ごとに参照写真を登録 → 面色ベースで照合。縞模様・斑点による誤認識を抑える設計。

設定: Edge Function エンドポイント

Supabase Edge Function(chip-vision)をデプロイ後に発行されるURLと、設定したシークレット文字列を貼り付けてください。Anthropic APIキー自体はここには入力しません(サーバー側のSecretで管理)。

1色ごとの参照写真 + 額面

参照写真のコツ: 本番と同じ照明下・同じ場所で、チップ1枚の面をできるだけ大きく正面から撮影してください。パッケージ写真の流用や別環境での撮影は、照明差により精度が下がります。

2チップ撮影(横一列に並べる)

手順: 先にスマホの標準カメラアプリで撮影 → ここで選択してください。

横一列ルール: チップタワーは重ならないよう間隔を空けて並べてください。

2b⚖️ 計量モード(Phase 0・画像認識の代わりに重量から算出)

各色の1枚あたり重量(上の色設定欄)を登録した上で、実際に量った総重量(g)を入力してください。校正は1枚単体ではなく10〜20枚まとめて量って割り算すると精度が上がります。

🧪 実験: 側面高さ計測(AR風スケール較正)

正式な精算には使わないでください(検証専用)。チップの実寸を使って画面上でスケールを較正し、スタックの高さから枚数を逆算します。
